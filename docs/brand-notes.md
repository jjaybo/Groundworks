# J & E Professional Services Brand Notes

These notes summarize the logo usage PDF provided on June 4, 2026.

## Brand Name

J & E Professional Services, LLC

## Tagline

Your property, our priority!

## Logo Usage

- Official SVG installed in the app: `static/official-logo.svg`.
- Use the horizontal logo with tagline when there is enough room.
- Use the horizontal logo without tagline when vertical space is limited.
- Use the vertical logo for square or vertical layouts.
- Use the small-size logo for small print or compact website areas.
- Maintain clear space around the logo. The brand book defines the safe area around the logo as the height of the first letter used in the logo.
- Approved variations include logo on white, logo on corporate green, transparent background, black and white, and negative version.

## Colors

- Primary green: `#3B9E3E`, RGB `59, 158, 62`
- Secondary green: `#76BB78`, RGB `118, 187, 120`
- Design element 1: `#D8ECD8`, RGB `216, 236, 216`
- Design element 2: `#ECF5EC`, RGB `236, 245, 236`

## Typefaces

- Cassannet Bold
- Open Sans Bold

For the starter web app, the CSS names these fonts when available and falls back to common system fonts so the app still works before custom font files are installed.
